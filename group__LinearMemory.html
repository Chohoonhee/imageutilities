Detailed Description

Memory management for LinearMemory classes.

This handles the memory management for following linear memory classes:

  • LinearHostMemory
  • LinearDeviceMemory
  • TensorCpu
  • TensorGpu

The device memory classes can be easily passed to CUDA kernels using a special struct. This struct gives the possibility to not only access the data pointer of the image but also other useful information such as length of the object.

  • LinearDeviceMemory::KernelData
  • TensorGpu::TensorKernelData
